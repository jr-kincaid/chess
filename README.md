2. Document Openings.
3. Create utilities that help automate other goals.

## Forsyth-Edwards Notation (FEN)

This notation is based on the work of two Chess enthusiasts separated by time.

David Forsyth, a Chess player who was born in 1854, and would later go on to become the chess editor for the Glasgow Weekly Herald. Steven James Edwards was a computer scientist that lived until 2016. Steven would take David's work and form the Forsyth-Edwards Notation as later conceive Portable Game Notation (PGN) AND Extended Position Description. 

This is only the beginning of the rabbit hole.
